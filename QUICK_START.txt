                    🚀 HOW TO RUN IT

Step 1: Start Python Backend (CRITICAL!)
───────────────────────────────────────

Windows:
  1. Open Command Prompt
  2. cd video-model
  3. start-backend.bat
  4. Wait for: "Running on http://0.0.0.0:5000"

Mac/Linux:
  1. Open Terminal
  2. cd video-model
  3. chmod +x start-backend.sh
  4. ./start-backend.sh
  5. Wait for: "Running on http://0.0.0.0:5000"


Step 2: Start Next.js Website
──────────────────────────────

  npm run dev

  Wait for: "Ready in Xs"


Step 3: Test It!
────────────────

                    ❓ TROUBLESHOOTING

Problem: "Service unavailable" error
Solution: Make sure Python backend is running on port 5000
          → Check terminal for "Running on http://0.0.0.0:5000"

Problem: "Model not loaded" error
Solution: Ensure asl_model.pt exists in video-model/ folder
          → Try restarting backend

Problem: Connection refused
Solution: Check both servers running:
          → Backend on port 5000
          → Frontend on port 3000
